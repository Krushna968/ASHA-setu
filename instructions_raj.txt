PART 1: WHAT IS THIS PROJECT?
================================================================================

ASHA-Setu is a mobile app built for ASHA (Accredited Social Health Activist) 
workers in India. These are community health workers who go door-to-door in 
rural areas to track pregnancies, infant health, immunizations, etc.

This app helps them:
- Log in with OTP (SMS verification)
- See their daily dashboard with pending tasks
- Manage their assigned patients
- Log patient visits with health vitals (BP, weight, etc.)
- View their visit calendar
- Track medical supply inventory (medicines, kits)
- Learn new health modules (training videos/articles)
- Send messages to their district officer
- Access emergency contacts for quick dial
- Get help & FAQs
- Manage their own profile

The app has TWO parts:
1. "App frontend" folder = Flutter mobile app (Dart language)
2. "App backend" folder = Node.js server (JavaScript) â€” YOU WILL NOT TOUCH THIS

You will ONLY work inside the "App frontend" folder (for 1 screen) and the 
root folder (for documentation files).


================================================================================
PART 2: TECH STACK (so you understand the project for documentation)
================================================================================

Frontend:
- Flutter 3.x (Dart programming language)
- Material Design widgets
- Custom dark theme (defined in lib/theme/app_theme.dart)

Backend:
- Node.js + Express.js (REST API)
- Prisma ORM (database query builder)
- PostgreSQL database (hosted on Supabase cloud)
- AWS SNS for OTP SMS delivery
- JWT (JSON Web Tokens) for session authentication

Infrastructure:
- GitHub for version control
- Supabase for database hosting
- AWS for SMS delivery
- Vercel for admin web portal hosting (separate project)
